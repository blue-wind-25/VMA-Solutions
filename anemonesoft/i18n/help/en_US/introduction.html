Introduction

This application is developed mainly to help the validation of common analytical chemistry in industries and laboratories. However, this application also comes with few additional features.

Please consider donating to support our software development.

Features

  • Cross platform (implemented using Java 6 SE).
  • Graphical user interface:
    • Tab-based working area with one common spreadsheet.
    • Accordion-based setting panels for fast access to the available settings.
    • Toolbar and keyboard short-cuts for commonly used commands.
    • Common look-and-feel in all supported platforms (with slight variations).
    • Simple help system.
  • General capabilities:
    • Project-based data management (the same spreadsheet can be used to generate multiple plots and analysis processes).
    • Zooming plot in an independent window.
    • Printing plot and report.
    • Saving plot as image file (currently supports PNG, JPEG, and BMP).
    • Saving report as plain text or HTML file.
  • Supported plots:
    • Scatter-line plot with double Y-axis.
    • Box-whisker plot with double Y-axis.
  • Supported analysis:
    • Linearity:
      • First order regression with LOD, LOQ, and Fisher lack-of-fit test.
      • Second order regression.
      • Mandel Test.
      • Homogeneity.
    • Accuracy.
    • Precision (RSD and 1-Way ANOVA).
    • Robustness.
    • Accuracy Profile (Gonzáles and Rozet methods).
    • Z Score
    • Z' Factor
    • Grubbs Test

System Requirements

  • A recent-enough computer with an operating system supported by Java Platform Standard Edition 6 (Java SE 6).
  • Java SE Runtime Environment 6 (JRE) (runtime version 1.6.0_20 or newer is recommended).
  • A unique license data (key) for each host.
