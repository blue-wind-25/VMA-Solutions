Basics of the Graphical User Interface

Upon starting the application, one will be presented with the application splash-screen. After approximately three seconds, the splash-screen will disappear and the main application window will be displayed.

(click image to enlarge)

The Main Application Window

An empty application main window is shown below. Please note that the actual appearance of the window may differ slightly between platforms and operating systems.

(click image to enlarge)

Basically, the window can be divided into four major sections, from top to bottom:

  • The window-title area;
  • The menu area;
  • The toolbar area; and
  • The working area.

The window-title area belongs to the operating system. The menu area contains all the commands that can be used to perform tasks, such as creating a new project, opening an existing project, saving the current project, creating plot, performing analysis, and so on. The toolbar are contains short-cuts to some of the most commonly used commands. Finally, the working are is the area in which one will be able to enter and process data. By default, upon application start, the working area will only contain the Data Input tab (the spreadsheet).

The Toolbar Area

The toolbar area can be divided into three sections, from left to right:

  • The left-button area;
  • The middle-information area; and
  • The right-button area.

The left-button area contains buttons that act as short-cuts to some of the commands in the File menu. The middle-information area displays the path of the currently open project. The Data Input Tab

The data-input tab is the only tab that cannot be closed. One can use the spreadsheet in this tab to enter data. Simply click any cell to start entering data. Pressing Enter will move the cursor to the next row. One can also use arrows to navigate the cells. Press F2 to edit the content of the active cell. The cells only accepts numeric values (numbers).

(click image to enlarge)

Click on a cell and drag the mouse pointer to select cells. One can also select cells by holding the Shift key while pressing the arrows. The selected cells can be cut, copied, and cleared by using the commands from the Spreadsheet menu, the buttons from the toolbar, or the keyboard short-cuts. Please have a look at the commands in the Spreadsheet menu for the keyboard short-cuts associated with the commands.

One may enter both data (numbers) and comments in the cells. All non-number entries are assumed as comments. If comments are included in the input data range, they will be evaluated as if they were zeroes. If all cells in the first row (from all the selected rows) contain only comments, the row will be assumed as a header row.

One can double-click the data input tab to change its caption.

The Plot Tab

The plot tab is the tab where plots are generated and displayed. Basically, the tab can be divided into three major sections:

  • The setting-accordion section (the left part of the tab);
  • The caption-setting section (the bottom-right part of the tab); and
  • The plot-display section (the top-right-part of the tab).

The contents of each section may differ between plot types. The picture below depicts the appearance of a box-whisker plot tab:

(click image to enlarge)

One can use the setting-accordion section to specify the data range, data point (plot) style, axis scale, and some other settings. One can use the caption-setting section to set the plot captions. Finally, one can use the plot-display section to update the plot, zoom the plot in an external window, print the plot, and save the plot as image. Please note that changes in the settings will not be propagated to the plot until the update button is pressed.

One can double-click the plot tab to change its caption.

The Analysis Tab

The analysis tab is the tab where analysis are performed and results are displayed. Basically, the tab can be divided into three major sections:

  • The setting-accordion section (the left part of the tab);
  • The caption-setting section (the bottom-right part of the tab); and
  • The result (plot and report) section (the top-right-part of the tab).

The contents of each section may differ between analysis types. The picture below depicts the appearance of a 1st-order-regression analysis tab:

(click image to enlarge)

One can double-click the analysis tab to change its caption.

One can use the setting-accordion section to specify the analysis setting, data range, plot style, axis scale, and some other settings. One can use the caption-setting section to set the plot captions. Finally, one can use the result section to update the plot/report, zoom the plot in an external window, print the plot/report, and save the plot/report to file. Just like the plot tab, changes in the settings will not be propagated to the plot/report until the update button is pressed.

Unlike a plot tab, an analysis tab may display other results, in addition to the plot. Some analysis may even have no plot to display. One can choose which results to be displayed by clicking the tabs on the result section. In the case of the 1st-order-regression tab, the available choices are:

  • The regression-plot tab;
  • The residual-plot tab;
  • The report-text tab; and
  • The content-calculation tab.
