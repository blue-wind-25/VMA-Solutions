The Setting Panels

There are several setting-panels in the application. Each panel is meant to be used to configure a specific part of the process.

The Plot-Caption Setting Panel

Using this panel, one can configure:

  • The plot's caption;
  • The plot's sub-caption;
  • The plot's X-axis' caption;
  • The plot's left Y-axis' caption; and
  • The plot's right Y-axis' caption (if the plot has it).

In case there are multiple plots, then the plot-caption setting panel will have tabs in it.

VMA supports inserting symbols, italic, superscript, and underscript. Simply click one of the buttons on the right side of the panel to insert symbols, italic, superscript, or underscript when editing the captions. The picture below depicts the appearance of the insert-symbol dialog. Please note that the actual appearance of the dialog box may differ slightly between platforms and operating systems.

VMA also supports using the cells' values as the captions. To use this feature, simply put the cells' references (such as $[A1], $[A2], $[B1], $[B2], and so on) as the captions.

The Analysis Setting Panel

This panel would be the most simple setting panel. There are only two settings in this panel. The Statistical Confidence field defines how much error one can tolerate. In the picture below, it means 5% error is acceptable.

     

The Number of Determinations field is only used for drawing prediction bands on regression plots. The Tolerance Interval field is only used in precision (RSD) analysis. The λ field is only used in accuracy profile analysis.

The Input-Data-Range Setting Panel

This is the setting panel that connects the spreadsheet to the plot-drawing and analysis system. The appearance of this panel can be different for different types of plot and analysis. In a plot that supports both left and right Y-axis, there will be check boxes to define whether the particular series should use the right Y-axis. The default is unchecked (all using the left Y-axis).

  

Please note that, for the time being, the application does not support multiple columns for the same data range. In the above picture, the X Values is specified to use the data from spreadsheet cells A1 to A5. If one changed that data range into multiple columns such as A1-B5, F3-G5, and so on, no plot will be drawn and nothing will be analyzed.

The Data-Range-Caption Setting Panel

The X-axis tick values (labels) on most of the plots are generated automatically from the X-axis scale settings. However, there are plots that requires custom labels. One can specify those labels using this panel. The appearance and usage of this panel can be different for different types of plot and analysis. VMA also supports using the cells' values as the labels. To use this feature, simply put the cells' references (such as $[A1], $[A2], $[B1], $[B2], and so on) as the labels.

The Plot-Style Setting Panel

One can use this setting panel to specify the color, symbol types, and line types for the plot. The appearance if this panel can be different for different types of plot and analysis, but their functions remain the same.

     

The Axis-Scale Setting Panel

Selecting the correct axis scale would be crucial to produce a good plot. The application tries to find the best axis scale for the plot. However, one may want to change the axis scale to produce a more clean and clear plot. In a plot that supports both left and right Y-axis, there will be separated settings for each of the Y-axis.

  

The Miscellaneous Setting Panel

One can change settings in this panel to change the plot's background and text colors. If one would like the application to draw the origin line and the grid line on the plot, simply select the line style to anything but Invisible.
