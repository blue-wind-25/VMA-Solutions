The Dialog Boxes

The system will display dialog boxes when it needs further input from the users. Each dialog box usually corresponds to a specific action. Please note that the actual appearance of the dialog boxes may differ between platforms and operating systems.

The Open-Project Dialog

This dialog box will be displayed when the user decides to open an existing project. Despite its main function to select a project file to be open, one can navigate the directories, create new directories, and rename files/directories from within this dialog box.

The Save-Project-As Dialog

This dialog box will be displayed when the user decides to save the current project to a new file. Its primary function is for specifying the name of the project file. However, one can also navigate the directories, create new directories, and rename files/directories from within this dialog box.

The Page-Setup Dialog

The primary function of this dialog is to perform an application-wide page setup. All future printing action will use the page settings specified in this page. Please note that the settings will not be saved to disk. Therefore if one restarts the application, all settings will revert to their original values.

The Zoom-Plot Dialog

Sometimes, it could be convenient to view the plot in a bigger, stand-alone window. This dialog box will be displayed each time a user click on the Zoom plot button on the working area.

The Print Preview and Print Dialogs

These dialog boxes will be displayed before any print job is sent to the printer.

The Select-Image-Quality Dialog

This dialog box will be displayed when the user wants to save a plot as an image file. One can use this dialog to specify the resolution of the resulting image.

The Save-Plot-As Dialog

This dialog box will be displayed after the user selects the image resolution. Its primary function is for specifying the name and type of the image file. However, one can also navigate the directories, create new directories, and rename files/directories from within this dialog box. The application currently supports PNG, JPEG, and BMP image types.

The Save-Report-As Dialog

This dialog box will be displayed when the user decides to save the report into file. Its primary function is for specifying the name and type of the report file. However, one can also navigate the directories, create new directories, and rename files/directories from within this dialog box. The application currently supports plain text and HTML as the report-file types.
